# Migratory Dawn — Codex Context Package

This package contains the project context needed for Codex to continue development without reading the original ChatGPT conversation.

## Files
- `AGENTS.md` — operating instructions for Codex
- `docs/PRD.md` — frozen product requirements
- `docs/DESIGN.md` — visual and interaction direction
- `docs/TECHNICAL.md` — architecture and implementation plan
- `docs/PROGRESS.md` — completed work, current blocker, next milestone
- `docs/DECISIONS.md` — frozen decisions and rationale
- `docs/CODEX_FIRST_PROMPT.md` — first instruction to send to Codex

## Use
1. Repair the npm certificate issue with Codex.
2. Create the `migratory-dawn` repository.
3. Copy these files into the project root, preserving paths.
4. Ask Codex to read all files before making changes.
5. Keep `PROGRESS.md` and `DECISIONS.md` updated.
